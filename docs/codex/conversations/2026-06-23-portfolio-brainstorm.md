# Conversation Summary - Portfolio Website Brainstorm

日期：2026-06-23

## 目标

为王柯杰从零规划一个 React + Vite 个人作品集网站，用于内容运营、新媒体运营及相关运营实习岗位求职。

## 已确认方向

- 网站偏个人作品集，而不是电子简历。
- 目标岗位偏内容/新媒体运营，同时保留通用运营适配性。
- 视觉风格暗色、高级、克制、有科技感，参考 Apple 的清晰叙事。
- 页面 PC 优先，移动端暂不做完整设计。
- 不展示手机号，Contact 展示邮箱和抖音入口。
- 抖音入口并入 Contact，不进入主导航。
- 配色采用 Harbor Graphite，Pure Silver Blue 作为备选。

## 已确认结构

- Hero：城市夜景/建筑线条背景，底部滚动摄影作品带。
- Hero 滚动作品只放摄影作品。
- Profile：About + Profile 融合，左侧头像和摄影作品叠层。
- Works：6 个主卡，包括摄影、抖音、设计物料、LSEG、融媒体、校园增长。
- Strengths：能力卡需要弹层。
- Works：项目卡需要弹层。
- 摄影作品点击后需要完整大图查看器。
