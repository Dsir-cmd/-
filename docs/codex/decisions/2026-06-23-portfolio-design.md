# 个人作品集网站设计说明

日期：2026-06-23

## 1. 项目定位

本网站是王柯杰用于运营类实习求职的个人作品集 Demo，核心目标不是复刻简历，而是让招聘方在短时间内记住一个清晰形象：

> 一个偏内容与新媒体运营方向，具备摄影审美、视觉设计、内容传播、活动执行和 AI 提效能力的复合型运营候选人。

目标岗位以内容运营、新媒体运营为主，同时保留品牌运营、市场运营、产品运营相关岗位的通用性。

## 2. 总体风格

网站采用暗色、克制、高级、科技感的视觉方向，参考 Apple 式清晰叙事和沉浸式产品页面，但不直接模仿 Apple 官网，也不做模板化作品集。

关键词：

- 专业
- 克制
- 冷静
- 视觉化
- 中英文并行
- 作品集感强于简历感

设计避免：

- 荧光绿潮玩感
- 过度模板化卡片网格
- 纯摄影师网站倾向
- 纯电子简历倾向
- 大量装饰性渐变和无意义动效

## 3. 配色系统

首选配色为 `Harbor Graphite / 港口石墨`，由用户摄影素材中的城市夜景、港口、建筑线条、航空冷灰和暖色灯光提取而来。

| 角色 | 色值 | 用途 |
|---|---:|---|
| 主背景 | `#05070A` | 页面底色、全局暗场 |
| 次级背景 | `#0B1118` | 区块背景、导航背景 |
| 卡片背景 | `#111821` | 项目卡、能力卡、弹层面板 |
| 主文字 | `#F3F7FA` | 标题和重要正文 |
| 辅助文字 | `#9AA7B3` | 说明文字、标签、元信息 |
| 主点缀 | `#8EC5FF` | 主要按钮、链接、焦点状态 |
| 次点缀 | `#F2A65A` | 关键成果数据、少量高价值强调 |
| 线条边框 | `rgba(180, 200, 220, 0.16)` | 分隔线、低对比边框 |

备选配色为 `Pure Silver Blue / 纯冷银蓝`，如果首选方案在实际素材中显得过暖或过复杂，可退回更冷静的银蓝方案。

## 4. 字体与标题系统

整体采用中英文并行的表达方式，提高作品集的国际化和设计感。

示例：

- `WANG KEJIE / 王柯杰`
- `Content & New Media Operations Intern / 内容与新媒体运营实习生`
- `PROFILE / 关于我`
- `SELECTED WORK / 精选项目与经历`
- `CORE STRENGTHS / 个人优势`
- `CONTACT / 联系我`

排版原则：

- 英文主标题负责视觉张力。
- 中文副标题负责识别和语义清晰。
- 大标题保持简短，不写长句。
- 正文偏作品集语气，避免简历套话。

## 5. 页面结构

首版为 PC 优先的单页式作品集 Demo，版心控制在约 `1700px`，暂不做完整移动端设计。

页面包含 5 个主要区块：

1. `Hero / 首页`
2. `Profile / 关于我`
3. `Selected Work / 精选项目与经历`
4. `Core Strengths / 个人优势`
5. `Contact / 联系我`

导航栏固定为：

- `Profile / 关于我`
- `Works / 精选项目`
- `Strengths / 个人优势`
- `Contact / 联系我`

抖音入口不放入主导航，统一并入 Contact 区域。

## 6. Hero 首页设计

Hero 使用全屏沉浸式结构，参考用户提供的 Hero 截图中的效果关系，而非其具体设计风格。

核心元素：

- 全屏摄影背景图
- 暗色遮罩，保证文字可读
- 顶部悬浮胶囊导航
- 左中区域大标题
- 一句定位文案
- 联系按钮或作品入口按钮
- 底部横向自动滚动摄影作品带

已确认：

- Hero 背景采用城市夜景/建筑线条方向。
- Hero 滚动作品带只放摄影作品，不混入商业拍摄、设计物料或内容运营案例。
- 滚动摄影作品尽量多放，但需按视觉节奏和画面比例筛选。
- 点击滚动作品后打开大图查看器。
- 横图在首页可裁切为统一缩略图，点击后必须展示完整大图。

候选 Hero 主视觉方向：

- `明日新城67.JPG`
- `洋山港夜景.JPG`
- 其他城市夜景、建筑线条、港口冷色照片

航空摄影可进入摄影作品流和摄影项目详情，但不作为首版 Hero 主背景，以免第一印象偏离运营求职定位。

## 7. Profile 关于我设计

该区块融合 `About` 与 `Profile`，不直接使用传统 `Work Experience` 逻辑。原因是用户目前仍是学生，单纯强调工作经历会削弱校园运营、内容创作和视觉能力的综合优势。

建议标题：

`PROFILE / 关于我`

内容结构：

- 左侧：头像 + 摄影作品叠层主视觉
- 右侧：个人定位、关键信息、成果数据
- 下方：经历时间线

左侧视觉方向：

- 以用户本人头像或职业照建立信任感。

- 当前 `F:\作品材料\头像个人照片` 暂无素材，后续需补充。

右侧信息建议：

- 定位：内容与新媒体运营实习生
- 背景：上海海洋大学市场营销本科在读，2027 届
- 可实习：2026 年 6 月起，每周 5 天，连续 6 个月及以上
- 所在地：上海
- 联系方式：邮箱，不展示手机号

核心数据建议：

- `10+` 商业拍摄

## 8. Selected Work 精选项目与经历

该区块不是单纯作品图片墙，而是“作品 + 经历 + 能力证据”的核心展示区。

首版确认 6 个主卡：

1. `Photography Works / 摄影作品`
2. `Douyin Content Case / 抖音内容案例`
3. `Design Materials / 设计物料`
4. `LSEG Internship / 产品与市场支持`
5. `Campus Media Operations / 融媒体内容运营`
6. `Campus Growth & Community / 校园增长与社群运营`

设计策略：

- 前 3 个偏视觉化，证明审美、设计和内容表达。
- 后 3 个偏运营化，证明组织、执行、沉淀和业务支持。
- 每张卡片可点击打开详情弹层。
- 卡片可采用大尺寸不规则布局，但保持克制、清晰和 PC 端浏览效率。

1. `Content Planning & Distribution / 内容策划与传播`
2. `Photography & Visual Storytelling / 摄影视觉表达`

4. `Campaign & Community Operations / 活动运营与社群触达`

已确认交互：

- 首版重点让 3 个更差异化能力可点击展开：内容策划、摄影视觉、设计物料。

详情弹层内容结构：

- 能力说明
- 代表经历

各能力证据来源：

必须包含：

- 抖音主页入口

不包含：

可选：

联系方式建议文案：

## 11. 交互与动效

首版交互范围：

- 导航锚点滚动
- Hero 摄影作品自动滚动
- 摄影作品大图查看器
- Works 项目详情弹层
- Strengths 能力详情弹层
- Contact 邮箱复制或跳转邮件客户端
- 抖音主页外链

动效原则：

- 慢、轻、克制
- 不用夸张弹跳
- 不让动效影响阅读
- 内容默认可见，不依赖动画加载后才出现
- 需要考虑 `prefers-reduced-motion`
